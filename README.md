# Bladez N Fadez — Calgary's Premium Barbershop

A modern, responsive website for Bladez N Fadez, a premium barbershop in Skyview Ranch, Calgary.

## Features
- **Dynamic Home Page**: Highlighting unique selling points and customer reviews.
- **Service Catalog**: Detailed list of services with booking CTAs.
- **Barber Profiles**: Meet the craftsmen behind the cuts.
- **Project Gallery**: Filterable masonry grid showing the latest work.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile.

## Local Development
1. Clone the repository.
2. Run `npm install` (optional, for local server).
3. Run `npm start` to view the site at `http://localhost:3000`.

## Deployment to Render (Static Site)
To deploy this project to Render:
1. Create a new **Static Site** on [Render](https://render.com/).
2. Connect your GitHub/GitLab repository.
3. Use the following settings:
   - **Build Command**: `(Leave Blank)`
   - **Publish Directory**: `.`
4. Click **Create Static Site**.
